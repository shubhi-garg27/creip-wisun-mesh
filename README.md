# CREIP-WiSUN-Mesh

### Cognitive RF Environment Intelligence Platform

An intelligent Wi-SUN mesh networking platform built using Silicon Labs EFR32 devices for adaptive routing, RF-aware communication, and resilient IoT infrastructure.

---

## About The Project

CREIP is designed to make wireless IoT networks smarter and more reliable in environments where communication quality constantly changes due to interference, congestion, or weak signal conditions.

Instead of using traditional static routing, the platform continuously studies network behaviour and dynamically selects better communication paths using edge-AI based decision making.

The system is capable of:

* identifying unstable mesh links,
* predicting communication degradation,
* rerouting traffic automatically,
* and recovering from node failures without manual intervention.

The project focuses on building a self-learning wireless infrastructure for future industrial and smart-city applications.

---

## Why CREIP?

Modern IoT deployments often face:

* unstable communication,
* packet drops,
* increased latency,
* and poor scalability in dense RF environments.

CREIP addresses these issues by combining:

* Wi-SUN mesh networking,
* predictive routing,
* RF monitoring,
* and lightweight AI inference directly on edge devices.

This helps create a more stable, adaptive, and energy-efficient wireless ecosystem.
